# Superstore Sales Analysis

## Project Overview

This project presents an end-to-end Data Analytics workflow using the Superstore retail dataset. The analysis was performed using SQL, Microsoft Excel, and Power BI to uncover business insights related to sales, profit, customers, products, and regional performance.

## Objectives

* Analyze sales and profitability trends
* Identify top-performing and underperforming products
* Understand customer and regional performance
* Build interactive dashboards for business decision-making

## Tools & Technologies

* SQL (SQLite)
* Microsoft Excel
* Power BI
* DAX
* Git & GitHub

## Project Structure

### Dataset

Contains the Superstore dataset used for analysis and dashboard creation.

### SQL Analysis

Business-focused SQL queries covering:

* Sales Analysis
* Profit Analysis
* Customer Analysis
* Regional Analysis
* Aggregations and Reporting

### Excel Analysis

Interactive Excel dashboard featuring:

* Pivot Tables
* Pivot Charts
* KPI Tracking
* Slicers

### Power BI Dashboard

Interactive 3-page dashboard:

1. Executive Dashboard
2. Product Analysis
3. Customer & Regional Analysis

## Key Insights

* Technology generated the highest sales among product categories.
* Profitability varied across markets and regions.
* A small group of products and customers contributed significantly to total sales.
* Sales showed consistent growth across the analyzed period.
