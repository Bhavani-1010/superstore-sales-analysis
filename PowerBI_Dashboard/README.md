# Power BI Dashboard

## Overview

This folder contains the Power BI dashboard developed using the Superstore retail dataset. The dashboard provides interactive visualizations and business insights related to sales, profit, products, customers, and regional performance.

## Dashboard Pages

* Executive Dashboard
* Product Analysis
* Customer & Regional Analysis

## Features

* KPI Cards
* DAX Measures
* Interactive Slicers
* Sales and Profit Analysis
* Customer and Regional Insights

## Files Included

* Power BI Dashboard (.pbix)
* Dashboard Report (.pdf)
* Dashboard Screenshots

## Tools Used

* Power BI
* DAX
* Data Visualization
