# Excel Analysis

This folder contains the Excel-based analysis of the Superstore dataset.

## Workbook Contents

The Excel workbook includes the following sheets:

* **Raw_Data** – Original dataset used for analysis.
* **Pivot_Analysis** – Business analysis performed using Pivot Tables and Pivot Charts.
* **Business_Insights** – Key findings, business problems identified, and recommendations.
* **Lookup_Practice** – Demonstration of VLOOKUP and XLOOKUP functions.
* **Dashboard** – Interactive dashboard with KPI cards, charts, and slicers.

## Skills Demonstrated

* Data Cleaning and Formatting
* Excel Tables
* Pivot Tables
* Pivot Charts
* VLOOKUP
* XLOOKUP
* Dashboard Design
* Business Analysis and Reporting

## Key Insights

* Technology category generated the highest sales and profit.
* Tables was identified as the only loss-making sub-category.
* Consumer segment contributed the highest sales.
* APAC market generated the highest profit.
* Sales showed consistent growth from 2011 to 2014.

This analysis was created as part of an end-to-end Data Analytics project using Excel, SQL, and Power BI.
